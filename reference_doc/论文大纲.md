# 拓扑-几何联合驱动的合并树节点简化方法研究

## 0. 论文定位

本文面向标量场可视化中的合并树简化问题。传统合并树或轮廓树简化方法通常侧重拓扑持久性、节点高度差、局部体积等指标，这类方法能够有效减少拓扑结构复杂度，但对增广合并树中大量普通节点的几何冗余刻画仍然不足。

本文关注的问题不是删除关键拓扑事件，而是在保留临界节点与主要拓扑结构的前提下，对拓扑边内部大量由增广过程产生的普通节点进行筛选。核心思想是：一条拓扑边内部通常对应同一拓扑阶段中的连续几何演化，其中许多相邻节点代表的等值面形态高度相似，可以被视为冗余采样；只有当等值面在几何形态、空间覆盖区域或原始网格截交结构上发生显著变化时，才需要保留对应节点。

因此，本文试图建立一种“拓扑约束下的几何感知简化”框架：拓扑结构决定哪些节点不能删、哪些节点属于同一条可简化链；几何与原始网格结构决定链上普通节点是否真正携带新的形态信息。

## 1. 引言

### 1.1 研究背景

标量场数据广泛出现在流场模拟、医学成像、地球科学和工程仿真等场景中。为了理解标量场中等值面的生成、合并、分裂和消亡过程，拓扑数据分析常使用合并树、分裂树、轮廓树等结构来描述标量场的全局拓扑演化。

合并树能够将复杂的三维标量场压缩为一棵树形结构，其中临界节点对应重要拓扑事件，弧或拓扑边对应等值面连通分量在两个拓扑事件之间的连续演化。为了支持更细粒度的分析，实际计算中常得到增广合并树，即在临界节点之间插入大量普通节点。这些普通节点有助于记录等值面沿标量方向的变化过程，但也会显著增加树的规模和可视化复杂度。

在当前实验数据中，原始合并树输出包含数百个节点，其中大量节点并非新的拓扑事件，而是拓扑边内部的中间采样点。如果直接显示或分析全部节点，会带来三个问题：

- 树结构过密，难以观察主要分支与关键演化阶段。
- 可视化结果中存在大量形态近似的等值面，造成认知冗余。
- 后续对比、交互、存储和传输成本增加。

因此，需要一种能够在不破坏主要拓扑结构的前提下，压缩增广合并树普通节点的方法。

### 1.2 现有简化思路的不足

传统拓扑简化通常依赖持久性或高度差等拓扑指标。这些指标适合判断临界点对的重要性，但当问题转向“同一条拓扑边内部的普通节点是否冗余”时，仅靠拓扑指标并不充分。原因在于：拓扑边内部的所有普通节点本身并不改变连通性，它们在拓扑意义上可能等价，但在几何意义上可能差异很大。

另一方面，单纯依赖体积、节点间标量差或采样间隔也存在局限。两个等值面可能体积相近，但形状已经从紧凑结构变成长条结构；也可能总体体积变化不大，但局部出现细颈、分叉趋势、空间偏移或结构延展。此时，如果只根据体积或标量间隔删除节点，就可能丢失人眼可感知的重要几何变化。

因此，合并树节点简化需要同时回答两个问题：

1. 哪些节点在拓扑上必须保留？
2. 在拓扑允许简化的链内部，哪些普通节点在几何上仍然重要？

这正是本文引入拓扑约束和几何度量的主要原因。

## 2. 研究动机

本文的核心动机可以从两个角度展开：几何角度和拓扑角度。二者不是并列孤立的，而是共同决定了合并树简化的合理性。

### 2.1 动机一：几何角度

从几何角度看，增广合并树中的普通节点可以理解为等值面演化过程中的离散采样点。沿着一条拓扑边，标量阈值不断变化，对应等值面或等值区域也随之连续变化。理论上，如果每一个采样点都对应明显不同的形态，那么这些节点都值得保留；但在实际数据中，相邻节点之间常常只是等值面的平滑移动、轻微膨胀或局部小扰动，许多节点并不提供新的几何信息。

也就是说，增广合并树存在明显的几何冗余：

```text
拓扑边上的密集普通节点
        ↓
大量相邻等值面形态相似
        ↓
可视化与分析中重复表达同一阶段的几何状态
```

仅保留临界节点虽然可以得到更简洁的拓扑骨架，但会丢失拓扑事件之间的几何演化过程。例如，在两个临界点之间，等值面可能经历从球状到细长状的变化，也可能在空间中逐渐延展到新的区域。这个过程没有产生新的拓扑事件，却可能对理解流场结构、物理过程或空间分布非常重要。

因此，本文希望在临界节点之间进一步选择少量“几何关键帧”。这些关键帧不一定对应新的拓扑事件，但对应等值面形态发生明显变化的位置。它们能够在较少节点数量下，重建出拓扑边内部主要的几何演化脉络。

几何动机具体包括以下几方面。

第一，等值面形状变化并不总是由拓扑变化触发。拓扑结构只记录连通性变化，而不记录形状、面积、空间跨度、弯曲程度或局部延展。两个等值面可以拓扑等价，但几何形态差异显著。因此，如果只保留拓扑临界点，会忽略拓扑事件之间的重要形态变化。

第二，相邻普通节点之间存在大量形态重复。增广节点往往来自细粒度采样或原始网格顶点顺序，节点数量由数据离散程度决定，而不一定由几何变化程度决定。高分辨率数据会产生更多普通节点，但其中许多只是更密集地采样同一段平滑变化。

第三，几何变化需要多尺度描述。体积变化可以捕捉整体膨胀或收缩，距离场互信息可以捕捉形状差异，活性子图可以捕捉等值面在原始网格中的截交结构变化。不同几何描述从不同层次衡量“两个普通节点是否真的不同”。

第四，合理的几何节点筛选能够提升可视化表达效率。简化后保留的节点不再是均匀或机械采样，而是更接近等值面演化过程中的关键状态。这样既减少重复等值面，又保留主要形态变化，使合并树可视化更适合分析与展示。

因此，从几何角度出发，本文的关键问题是：

> 在同一拓扑阶段内，如何识别等值面形态发生显著变化的节点，并删除形态近似的冗余节点？

### 2.2 动机二：拓扑角度

从拓扑角度看，合并树的主要价值在于刻画标量场中连通分量的出现、合并与消亡过程。临界节点对应拓扑事件，是树结构的骨架；普通节点则位于拓扑事件之间，通常不改变树的分支关系。

这意味着，合并树简化不能简单地把所有节点当作普通几何采样点处理。临界节点、分支节点、根节点和叶节点具有特殊拓扑意义，必须受到保护。否则，简化可能破坏合并树表达的基本拓扑关系，使树不再正确描述标量场的连通性演化。

因此，本文的第二个动机是：利用拓扑结构为几何简化提供约束边界。

具体来说，拓扑角度提供了三个关键约束。

第一，拓扑临界节点必须保留。极小值、极大值、鞍点以及分支结构节点对应连通分量的生成、合并或消亡，是合并树的语义核心。如果删除这些节点，即使几何上看似变化不大，也会破坏原始拓扑结构。

第二，简化应限制在拓扑边内部进行。两个临界节点之间的一条拓扑边表示同一拓扑阶段中的连续演化。在这一区间内，普通节点通常不改变连通关系，因此可以根据几何冗余程度进行筛选。换言之，拓扑结构先将全局问题分解为多个局部链问题，使几何比较更有意义也更可控。

第三，简化结果需要保持树结构可重建。删除普通节点后，原来的多段小弧需要重连为新的简化弧，并保留节点之间的上下游关系。最终输出不只是一个节点集合，而应该仍然是一棵可解释、可可视化、可继续分析的简化合并树。

从这个角度看，拓扑不是被简化的对象，而是简化过程的约束框架：

```text
拓扑结构决定可简化范围
几何度量决定链内保留节点
重建过程保证简化后仍是一棵合法树
```

这也解释了为什么本文不直接对所有等值面做全局聚类或全局相似性筛选。全局比较可能把不同拓扑分支上的几何相似等值面误认为冗余，但这些等值面在树结构中承担完全不同的拓扑语义。相反，本文只在同一拓扑链内部比较节点，使“几何相似”建立在“拓扑语义一致”的前提下。

因此，从拓扑角度出发，本文的关键问题是：

> 如何在保持合并树关键拓扑事件和分支结构不变的前提下，对拓扑边内部的普通节点进行几何感知压缩？

### 2.3 几何动机与拓扑动机的结合

几何角度强调“哪些普通节点真的带来了新的形态信息”，拓扑角度强调“哪些节点和结构不能被破坏”。二者结合后，本文的问题可以表述为：

> 给定一棵增广合并树，如何在保留临界节点和拓扑分支结构的前提下，从每条拓扑边的普通节点中选出少量几何代表节点，使简化树既保持拓扑正确性，又尽可能保留等值面演化过程中的主要几何变化？

这一问题具有明确的层次结构：

1. 拓扑层：抽取结构节点之间的拓扑链，确定简化范围。
2. 几何层：对链上候选节点构造等值面或局部区域描述。
3. 相似性层：比较候选节点与参考节点之间的几何/结构差异。
4. 决策层：根据阈值保留几何变化显著的节点。
5. 重建层：根据保留节点重建简化后的树结构。

## 3. 问题定义

设输入为一个标量场：

```text
f : M -> R
```

其中 `M` 是规则网格或非结构网格域。由该标量场得到一棵增广合并树：

```text
T = (V_T, E_T)
```

树节点集合 `V_T` 包括临界节点和普通增广节点。对于任意一条由两个结构节点连接的拓扑链：

```text
C = {v_0, v_1, ..., v_k}
```

其中 `v_0` 与 `v_k` 是结构节点，中间节点 `v_i` 是候选普通节点。目标是在满足以下条件下选出保留节点集合 `K_C`：

```text
{v_0, v_k} subset K_C subset C
```

约束一：拓扑端点必须保留。

约束二：保留节点顺序应与原始链一致。

约束三：被删除节点应与最近保留参考节点在几何或结构描述上足够相似。

目标是最小化冗余节点数量，同时保持链上主要几何变化：

```text
减少 |K_C|
同时保证相邻保留节点之间的几何差异不超过可接受范围
```

最终在所有拓扑链上执行该过程，得到简化后的合并树：

```text
T' = (V'_T, E'_T)
```

其中 `V'_T` 包含所有结构节点和被选中的几何代表节点。

## 4. 总体方法框架

本文采用统一的拓扑约束链上贪心框架。不同方法的差异主要在于如何度量两个候选节点之间的几何或结构差异。

整体流程如下：

```text
输入标量场与增广合并树
        |
        v
识别结构节点：临界点、分支点、根、叶
        |
        v
抽取结构节点之间的拓扑链
        |
        v
对每条链执行顺序贪心筛选
        |
        v
根据保留节点重建简化树
        |
        v
输出简化合并树与可视化结果
```

对每条链：

```text
keep = [v_0]
ref = v_0

for v_i in v_1 ... v_{k-1}:
    compute Dist(ref, v_i)
    if Dist(ref, v_i) >= epsilon:
        keep.append(v_i)
        ref = v_i

keep.append(v_k)
```

其中 `Dist` 可以由不同方法定义：

- 基于距离场互信息的 ISM 距离。
- 基于子水平集体积的 IoU 距离。
- 基于原始网格活性子图的结构距离。
- 基于 GNN embedding 的向量距离。

## 5. 方法一：基于 ISM 的几何形状相似性度量

### 5.1 方法动机

体积或标量间隔只能描述等值面演化的某一方面，无法完整反映形状变化。为了更直接地衡量两个节点对应等值面是否相似，可以将等值面转化为距离场，并使用信息论指标比较两个距离场的相似程度。

距离场描述了空间中每个采样点到等值面的距离，因此相比直接比较三角网格，它对局部采样差异和表面噪声更稳定，也能更全局地反映形态变化。

### 5.2 基本思想

对于节点 `v_i`，取其标量值作为阈值，构造局部等值面或局部边界，并生成距离场：

```text
D_i(x) = min_{y in S_i} d(x, y)
```

然后计算两个距离场的归一化互信息：

```text
NMI(D_A, D_B)
```

定义差异：

```text
Dist_ISM = 1 - NMI(D_A, D_B)
```

同时加入质心位移和包围盒尺度变化作为补偿项：

```text
DistPos
DistBbox
```

候选节点保留条件：

```text
Dist_ISM >= epsilon_ism
or DistPos >= epsilon_pos
or DistBbox >= epsilon_bbox
```

### 5.3 方法特点

优势：

- 能捕捉体积相近但形状不同的情况。
- 适合作为高保真几何简化指标。
- 结果较容易从等值面形态变化角度解释。

局限：

- 构建距离场和互信息计算成本较高。
- 参数较多，包括局部块大小、直方图 bins、距离阈值和几何补偿项。
- 对空局部区域或极小局部区域需要额外稳定性处理。

## 6. 方法二：基于体积 IoU 的快速几何变化度量

### 6.1 方法动机

在同一条拓扑链内部，等值区域通常呈现连续膨胀或收缩。此时，两个相邻节点之间的差异可以通过对应子水平集体积变化近似描述。体积度量计算简单、参数直观，适合作为快速 baseline。

### 6.2 基本思想

设两个阈值对应的子水平集体积为 `V_A` 与 `V_B`，若二者具有嵌套关系，则 IoU 可简化为：

```text
IoU(A, B) = min(V_A, V_B) / max(V_A, V_B)
```

差异定义为：

```text
Dist_vol = 1 - IoU(A, B)
```

当：

```text
Dist_vol >= epsilon_vol
```

时保留候选节点。

### 6.3 方法特点

优势：

- 计算速度快。
- 参数 `epsilon_vol` 具有直观解释。
- 在实验中表现出稳定的单调控制效果。

局限：

- 只关注体积，不直接关注形状。
- 对“体积相近但形态变化明显”的情况不敏感。
- 适合作为 baseline 或第一阶段初筛，不宜单独作为最终高保真方法。

## 7. 方法三：基于原始网格活性子图的结构相似性度量

### 7.1 方法动机

ISM 和 IoU 都从等值面或等值区域的几何属性出发，但等值面本质上是原始网格在某一阈值下的截交结果。若直接观察等值面穿过了哪些原始网格边，就可以得到一种更贴近离散数据结构的描述。

活性子图方法的动机是：两个节点对应的等值面如果穿过了相似的原始网格局部结构，那么它们在该拓扑阶段中可能表达相似的几何状态；反之，如果活性边集合或活性区域发生明显变化，则说明等值面已经扫过新的局部结构，应保留对应节点。

### 7.2 基本思想

对于原始网格边 `(u, v)`，若：

```text
f(u) <= t < f(v)
or
f(v) <= t < f(u)
```

则该边被阈值 `t` 截穿，称为活性边。所有活性边及其端点构成节点对应的活性子图。

第一版使用边集 Jaccard 距离：

```text
Dist_J = 1 - |E_A intersect E_B| / |E_A union E_B|
```

并加入质心和包围盒补偿项：

```text
Dist_J >= epsilon_j
or DistPos >= epsilon_pos
or DistBbox >= epsilon_bbox
```

### 7.3 方法特点

优势：

- 直接依赖原始网格结构，减少对等值面重建质量的依赖。
- 可解释为“等值面扫过的原始网格区域是否变化”。
- 易扩展到图编辑距离、谱图特征和 GNN 表示学习。

局限：

- 边集 Jaccard 对轻微位置偏移和局部扰动敏感。
- 活性子图提取和局部分割限制需要仔细设计。
- 参数尺度与 ISM/IoU 不同，需要单独校准。

## 8. 方法四：基于 GNN 的活性子图向量化

### 8.1 方法动机

直接比较活性边集合虽然直观，但过于离散。两个局部结构可能整体相似，却因为少量边未被激活或局部位置偏移导致 Jaccard 相似度明显下降。为提高鲁棒性，可以将每个活性子图编码为固定长度向量，在向量空间中比较结构相似性。

### 8.2 基本思想

对每个活性子图构造节点特征，包括标量值、阈值距离、中心化坐标、局部度数和分割标签编码。然后使用两层 GCN 做消息传递，并通过图级池化得到 embedding：

```text
Active subgraph
    -> node features
    -> 2-layer GCN
    -> mean/max pooling
    -> graph embedding z
```

节点间距离定义为：

```text
Dist_GNN = 1 - cosine(z_ref, z_candidate)
```

### 8.3 自监督训练

为了使 GNN 学到更稳定的结构相似性，可以对同一活性子图进行两次轻微扰动作为正样本，对 batch 中其他子图作为负样本，使用 InfoNCE / NT-Xent 进行对比学习训练。

训练目标：

```text
相同子图的不同增强版本 -> embedding 接近
不同子图 -> embedding 分离
```

当前训练日志显示，loss 从约 `4.06` 降至 `2.25`，负样本相似度从约 `0.98` 降至 `0.31`，说明训练能够增强不同子图之间的区分度。

## 10. 预期贡献

本文可以总结为以下贡献：

1. 提出一种拓扑约束下的合并树普通节点几何简化问题，将增广合并树简化从单纯拓扑临界点删除扩展到拓扑边内部的几何冗余筛选。

2. 从几何角度指出普通节点冗余的来源：拓扑边内部大量节点仅对应等值面的平滑移动或轻微形态变化，不一定携带新的几何信息。

3. 从拓扑角度提出简化约束：临界节点和结构节点必须保留，普通节点筛选应限制在同一拓扑链内部，以避免破坏合并树的拓扑语义。

4. 构建统一的链上贪心简化框架，可接入 ISM、IoU、活性子图和 GNN embedding 等多种差异度量。

5. 通过多组参数实验验证不同度量对节点保留率和简化强度的影响，并分析各方法在几何保真、计算效率和结构鲁棒性方面的差异。

## 12. 可直接放入引言的动机段落草稿

在增广合并树中，临界节点之间通常插入了大量普通节点，用于记录等值面沿标量方向的连续演化。虽然这些普通节点提高了树对原始标量场的采样密度，但也引入了显著冗余。许多相邻普通节点对应的等值面在几何形态上高度相似，仅表现为轻微膨胀、平移或局部扰动。直接保留全部普通节点会导致树结构复杂、可视化拥挤，并增加后续分析成本。因此，从几何角度看，有必要识别拓扑边内部真正代表形态变化的关键节点，将其作为等值面演化过程中的几何关键帧。

然而，合并树简化不能仅从几何相似性出发。合并树的核心语义来自临界节点和分支结构，它们描述了标量场连通分量的生成、合并和消亡过程。如果在全局范围内直接根据几何相似性删除节点，可能会破坏树的拓扑语义，甚至混淆不同拓扑分支上形态相似但语义不同的等值面。因此，从拓扑角度看，简化过程必须首先保护临界节点和结构节点，并将普通节点筛选限制在同一拓扑链内部。拓扑结构提供简化的约束范围，几何度量则决定链内哪些节点应被保留。

基于上述观察，本文研究拓扑约束下的几何感知合并树节点简化问题：在保持合并树关键拓扑事件与分支关系不变的前提下，从每条拓扑边内部选择少量几何代表节点，以较低节点数量保留等值面演化的主要形态变化。为此，本文构建统一的链上贪心简化框架，并从距离场互信息、体积 IoU、原始网格活性子图以及 GNN 图表示等多个角度设计节点差异度量。
