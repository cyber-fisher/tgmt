3.输入数据格式：

1. 原始体数据
文件示例：output/tangaroa_port_2.vti
需要字段：
- 标量场，例如 v
- SegmentationId
- RegionSize
- RegionSpan
- ImageData 的 Spacing、Origin、Extent

用途：
- Spacing 用于把体素个数转换为物理体积
- SegmentationId 可用于后续做 bounding box 补偿项
- ![image-20260321163730700](C:\Users\14273\AppData\Roaming\Typora\typora-user-images\image-20260321163730700.png)

2. merge tree 节点
文件示例：output/tangaroa_port_0.vtu
需要字段：
- NodeId
- Scalar
- CriticalType
- VertexId
- RegionSize

用途：
- NodeId 作为树节点主键
- Scalar 用于在 branch 内排序
- CriticalType 用于区分极小点、鞍点、极大点等关键点

3. merge tree 弧
文件示例：output/tangaroa_port_1.vtu
需要字段：
- downNodeId
- upNodeId
- SegmentationId
- RegionSize
- RegionSpan

用途：
- downNodeId / upNodeId 用于重建树结构
- RegionSize 作为 branch 上的局部体积增量
- SegmentationId 用于从 segmentation 体中取对应区域

3.实现思路：

新增一个类，增加以下函数：

1. readNodesFromVTU(...)
读取 node 输出，填充 NodeRecord 数组。

2. readArcsFromVTU(...)
读取 arc 输出，填充 ArcRecord 数组。

3. computeVoxelVolume(...)
从 vti 的 spacing 计算单体素体积。

4. buildAdjacency(...)
根据 arcs 给每个节点挂 incidentArcIds。

5. extractBranches(...)
从整棵树中提取全部 branch。

6. computeBranchPrefixVolumes(...)
对一条 branch 计算 prefixVolumes。

7. simplifyBranchByVolume(...)
按 eps_vol 对 branch 做贪心筛选。

8. rebuildSimplifiedTree(...)
根据每条 branch 的保留结果重建新的 node/arc 集合。

9. writeSimplifiedTree(...)
把简化结果写成 VTU/VTP/CSV/JSON 等可复用格式。
